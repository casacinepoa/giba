COMO ABRIR PROJETOS DE VERSÕES MAIS RECENTES
(do Adobe Premiere)

Quando se tenta abrir um projeto do Adobe Premiere que foi gerado por uma versão mais recente que a nossa, o programa invariavelmente responde que é impossível.

Mas:

(a) todo projeto do Premiere (.prproj) é um arquivo de texto, codificado em XML, e comprimido sem perdas no formato GZIP;

(b) a principal diferença entre os projetos de diferentes versões é um código colocado no arquivo APENAS para impedir que ele seja aberto por versões mais antigas do programa;

(c) entendendo isso, este código pode ser modificado, o que permite abrir o projeto.

PROCEDIMENTO:

(1) Com o programa 7-zip ou equivalente, extraia o arquivo-texto que está dentro do projeto. Para isso, selecione o projeto e clique em EXTRACT (extrair).


Na pasta de destino selecionada, o programa criará uma pasta com o mesmo nome do projeto (mas sem a terminação), e dentro dela um novo arquivo com o mesmo nome (também sem terminação). Perceba que o arquivo descomprimido é quase 20 vezes maior que o projeto original.


(2) Num programa de edição de texto simples (Notepad para Windows, Text Editor para Mac, Kate para Linux, etc.), abra o novo arquivo, já expandido.

Na quarta linha de texto, está o código da versão do Premiere em que o projeto foi criado. Repare que não é o nome oficial da versão, mas um código interno. No caso: 38.


(3) Modifique este código para corresponder à sua versão do Adobe Premiere.

(Para isto, você vai ter que abrir um projeto qualquer criado por esta sua versão. No caso, vou modificar para 33, que corresponde à minha versão, de 2017.)


(4) Salve o arquivo.

(5) Recomprima o arquivo modificado, com o mesmo programa (7-Zip) que foi usado para abri-lo. Para isso, selecione o arquivo e clique em ADICIONAR (Add).


O 7-Zip vai abrir um menu com opções.

(6) Selecione o formato de arquivo "gzip" e mantenha as outras opções no padrão.


Na mesma pasta, vai ser criado um novo arquivo comprimido, com o mesmo nome e a terminação .gz


(7) Atualize o nome do arquivo e modifique a terminação para .prproj


Este projeto é igual ao original, e pode ser aberto pela versão mais antiga do Premiere.
