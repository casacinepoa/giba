Byte - Unidade básica de memória de um computador, correspondente a um conjunto de 8 bits; pode assumir 256 valores diferentes (2 x 2 x 2 x 2 x 2 x 2 x 2 x 2). Também é uma unidade de espaço de memória.

1 kB (kilobyte) = 1024 bytes
1 MB (megabyte) = 1024 kB (milhão de bytes)
1 GB (gigabyte) = 1024 MB (bilhão de bytes)
1 TB (terabyte) = 1024 GB (trilhão de bytes)
1 PB (petabyte) = 1024 TB (quatrilhão de bytes)
1 EB (exabyte) = 1024 PB (quintilhão de bytes)

Arquivos - são seqüências de bits, normalmente organizadas em bytes, que podem ser entendidas pelo computador como um conjunto autônomo e funcional de dados. Existem vários tipos de arquivos, conforme o tipo de dados que eles armazenam (embora, no fim das contas, todos eles sejam seqüências de zeros e uns):

Programas - são os arquivos que armazenam instruções para o computador: instruções sobre como lidar com dados de outros arquivos, como transferir dados de um setor de memória a outro, da memória para a tela do computador ou para a impressora, etc.; e ainda instruções sobre como agir em relação a novas instruções geradas pelo usuário, através do teclado, do mouse, etc. Os arquivos de programas também podem ser classificados em: arquivos executáveis, bibliotecas, arquivos do sistema operacional, etc.

Arquivos de dados - são os arquivos que apenas armazenam dados, para que estes sejam manipulados pelos arquivos de programas, de acordo com as instruções do usuário. São os arquivos de texto, ou de imagem, de som, de imagem em movimento, etc.
