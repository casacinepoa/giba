ORGANIZAÇÃO DE UM PROJETO

Pastas

- Uma pasta para cada tipo de material:
/ brutas de imagem;
/ brutas de som;
/ clipes sincronizados;
/ timelines;
/ músicas;
/ ruídos;
/ offs;
/ cartelas;
/ imagens de arquivo;
/ animações;
/ etc.

- Uma pasta para o material de cada cena.

- Pastas dentro de pastas no caso de o número de cenas ser muito grande.

Timelines

- Um timeline para cada cena montada no primeiro corte,
além de timelines extras para montagens alternativas.

- Um timeline geral para consolidar o primeiro corte.

- A partir daí, um novo timeline para cada corte, além de alguns extras para algumas cenas que precisem ser repensadas isoladamente.

Nomes

- Os nomes devem ajudar a localizar as coisas: um mês depois, um ano depois; ou amanhã, pelo colega que teve que te substituir por algum motivo.

Projeto

- Todo o material dentro de um único projeto.

- Nome do projeto = nome do filme?

- Localização: onde está o arquivo do projeto?

- Becape diário do projeto em outro disco, se possível em outro endereço.

Autosaves

- Pasta específica, normalmente dentro da mesma pasta onde está o projeto.

- Nas configurações, pode definir a frequência e o máximo de versões.
